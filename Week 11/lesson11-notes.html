Lesson 23: Controlling Flow with Conditions and Loops

What You’ll Learn in This Lesson

  • How to test conditions with the if statement
  • How to use comparison operators to compare values
  • How to use logical operators to combine conditions
  • How to use alternative conditions with else
  • How to create expressions with conditional operators
  • How to test for multiple conditions
  • How to perform repeated statements with the for loop
  • How to use while and do...while loops
  • How to create infinite loops (and why you shouldn’t)
  • How to escape from loops and continue loops
  • How to loop through an array’s properties

Controlling the flow in which code executes is very important for programming code, and JavaScript is no different. The most common flow control statement is the if statement, which evaulates a boolean expression, also called a conditional expression, and execute the block of code within the statement if it evaluates to true. Conditional expressions are built with conditional operators, and multiple conditional expressions can be chained together with logical operatorsas one complex conditional expression. You can use the else keyword to execute code when the conditional expression in the previous if statement is evaulated to false. There are shorthand conditional expressions that you can use for simple if-else statements, which are useful if you want to assign a variable to a certain value depending on the conditional expressions.

If you need to control flow of code for multiple conditions (>2 conditions), you can use else if to add conditions besides the initial if and else statements. As an alternative to the if-else statements, you can the switch to combine multiple if-else statements into one block of code, given they share a variable or expression in common. Another important way to control the flow of code is the loop, and the for keyword allows to execute a block of code a specified number of times. Another set of loops, the while and the do...while execute an indefinite of times, only stopping when a conditional expression is false. The difference between the while and do...while loop is that the latter loop executes the block of code at least once before checking a conditional expression to execute the code again.

You should be careful with loops, especially with while and do...while loops, because if the condition expression isn't properly define, then you'll have an infinite loop. The keyword break helps you exit a loop statement, regardless of whether the condition expression has become false. continue is a keyword that allows you to start at the top of the loop block, regardless of other code below it. Another type of loop allows you to use the for statement to navigate properties of an object. Loops are commonly used to traverse arrays.

Lesson 24: Responding to Events and Using Windows

What You’ll Learn in This Lesson

  • How event handlers work
  • How event handlers relate to objects
  • How to create an event handler
  • How to detect mouse and keyboard actions
  • How to use onclick to change the appearance of <div>
  • How to access and use the window object hierarchy with JavaScript
  • How to delay a script’s actions by using timeouts
  • How to display alerts, confirmations, and prompts

A script that you create and use to detect and respond to an event, like a user clicking a button, is generally referred to as an event handler. Event handlers can be directly embedded as an attribute to a web element and can also call functions in JavaScript. To add multiple functions to trigger in response to an event, you can trigger them using the addEventListener method, which has several events that the object can "listen" for, from the DOM object. A built-in object, event, can help you modify the listening event such that a function or code only triggers when certain keys are pressed, or on a certain part of the page. Common events for which you can add event listeners is moving the mouse over a web element, clicking on a link, pressing a key, and loading a webpage.

A common command to execute in reponse to an event is changing the appearance of a web element in a div, such that certain certain context is shown or hidden when clicked. The Window object allows you to change the properties of the window that a web page is opened in, including closing windows and opening new windows. One use case of the Window object is resizing and moving the windows using the moveTo, moveBy, resizeTo, and resizeBy methods. Another useful method of the Window object is the setTimeout method, which allows to delay the execution of code after a defined amount of time. Finally, there are methods for providing dialog boxes to simply show messages, confirm a user event, and to prompt the user for text input.

JavaScript Conditions, Loops, and Event Handlers Reference

What is a conditional statement?

A conditional statement is a line or block of code that only executes when a certain criteria is met, usually a comparison of numerical values.

The conditional operators

  • == — Is equal to
  • != — Is not equal to
  • < — Is less than
  • > — Is greater than
  • >= — Is greater than or equal to
  • <= — Is less than or equal to

An example of multiple conditional expressions

Multiple conditional expressions can be chained together for a single conditional statement using the logical operators && and ||. An example is shown below. 


    if ((phone == "") || (email == "")) {
      alert("Something Is Missing!");
    }

A use case of loops

A use case of loops is to repeatedly prompt the user to enter information until the user enters a special character to indicate that they finished entering information.

What is an event handler?

An event handler is a script that is executed in response to an event, such as user moving the mouse over a certain location, or clicking on something.

"Click" event handler example

The following example shows a function mousealert that is triggered when a web element with an id of link1 is clicked on:

document.getElementById("link1").addEventListener("click", mousealert);

How can you make your page listen for keystrokes?

You can have your webpage listen for keystrokes by adding an event listener for keypress or keydown and defining a function with a conditional statement that looks for the desired keystroke. event.key can be used to "listen" for the desired key.

The properties of the Window object

  • window.closed — Indicates whether the window has been closed. This makes sense only when you’re working with multiple windows because the current window contains the script and cannot be closed without ending the script.
  • window.opener — In a new window opened by a script, provides a reference to the window containing the script that opened it.
  • window.outerHeight and window.outerWidth — Specify the height and width, respectively, of the outside of a browser window.
  • window.screen — Provides a child object that stores information about the screen the window is in—its resolution, color depth, and so on.
  • window.self — Acts as a synonym for the current window object.
