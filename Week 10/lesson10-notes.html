Three important aspects of JavaScript programming are variables, strings, and arrays. Variables can store data and have several rules regarding its naming, including only having letters, numbers and the underscore character, and starting with a letter or the underscore character, and fitting variable name on one line. Where variables can be used is based on scope: global variables can be called anywhere, but local variables can only be used within a function that the are crated in. While using var is optional for global variables, it is a good practice to use var for declaring all variables. There are several ways to incrementing or decrementing variables, including using +=/-= and ++/--.

JavaScript uses common arithmetic operators and the modulo operator to perform operations, and it operates by arithmetic precendence rules. The most common data types used in JavaScript are numbers boolean, string, and null. A JavaScript variable can store more than one type of data after it's declared, i.e. it can store a number and later store a string. JavaScript usually handles conversion between data types as necessary, but there are some methods, parseInt and parseFloat, that converts a number stored as a string, as a numerical value. String variables can be declared with either var or the new keyword, which is use to create new objects.

The addition operator, +, can be used to combine, or concatenate, strings, as well as the increment operator +=. Common methods used with a string object include the length method, which outputs the number of characters in a string, toUpperCase and toLowerCase, which converts all characters to upper case and lower case respectively, and substring, which accesses a certain part of a string. Other notable string methods include charAt, which returns the character at a specified index, and indexOf, which returns the index of a specified substring in a string. Arrays are a type of object consisting of a group of data items, including numbers or strings. You can declare a new array with the new word, or simply specify a new variable name and assign the array's contents within brackets ("[]").

Like strings, arrays have a length method, which gives you the number of items, or elements, in the array. You can access an array element of specifying the name of the array and the index of the element in brackets. For string arrays, you can use split to take a string with spaces and split it into a string array for all strings between spaces. join allows you to join elements of a string array into one string. Another array method, sort allows you to arrange an array in alphabetical order.

To sort numerical arrays, you insert a number comparison function so the sort function knows if one number belongs before or after another. Speaking of functions, they are a group of JavaScript statements that are grouped together and be called by its name. Functions can have parameters, which are variables that are passed from the main HTML page into the function for operations. Functions can also have return values, which will be substituted for the calling function.

Besides DOM objects and the built-in objects like strings and arrays, you can also define custom objects. You define a function, referred to as a constructor in this context, to create a custom object. Any parameters defined in the function can be used initialize the values of the object's properties. If you want the parameters and the property's name to be the same, you must use the this keyword to distinguish the parameters of an object and the properties of the object. To define a method for an object, you must define a separate function then assign the function to a method of the object.

By using the prototype keyword, you can define a new function or variable and assign it as a new variable and method of an existing object. An important built-in JavaScript object is the Math object. Some methods included in the Math object include the ceil, floor, and the round methods, which rounds up, rounds down, and rounds a number to the nearest integer respectively. Another important method is the random method, which generates a random number between 0 and 1. Another important object for JavaScript is the Date object which can store dates in several formats. There are many methods for setting dates, accessing parts of a date object, and converting date object to strings.

JavaScript Variables, Strings, and Arrays Reference

Rules for choosing variable names

  • Variable names can only include letters of the alphabet, numbers, or the underscore character.
  • The first character of the variable name must be either a letter or an underscore.
  • Variable names are case sensitive; for example, totalnum, Totalnum, and TotalNum are interpreted as separate variable names.
  • There is no official limit on the length of a variable name, but it must fit on one line.

Global and local variables

Global variables can be called anywhere, but local variables can only be used within a function that the are crated in.

Ways to assign value to variables

  • Using the keyword var
  • Using the keyword let
  • Using the keyword const
  • Using no keyword and assigning a value to a variable name (for global variables)

Ways to increment and decrement variables

  • Using the assignment operator to take a variable and add or subtract a new value (x = x + 1, x = x - 1)
  • Using the shorthand operators +=/-= which are equivalent to the above value assignment examples respectively (x += 1, x -= 1)
  • Using the shorthand operators ++/-- which are equivalent to incrementing or decrementing a variable by 1 only, respectively.

What is a string object?

A string object is a group of data and methods associated with a string, which is a group of characters.

What is an array?

An array is a numbered group of data items of the same data type.

What is a function?

A function is a group of statements in a block that can be called and reused multiple times in code.

How to return a value from a function

To return a value from a function, you use the return keyword as the last line in the function, followed by the value you want to return. You'll need to pass at least one value as a parameter in a function to return a value.

What is an object?

A object is a group of data items, which can be of multiple data types, and functions that are usually related together logically.

What is a Math object?

A math object is a group of common mathematical constants, like pi, and operations, like rounding and generating random number.

What is a Math object?

A math object is a group of common mathematical constants, like pi, and operations, like rounding and generating a random number between 0 and 1.

How to generate a random number between 1 and n?

The following code allows you to generate a random number between 1 and n:

Math.ceil(Math.random() * n);

or

Math.floor(Math.random() * n) + 1;

What is a Date object?

A date object is a group of date properties and methods that allows you to store dates in multiple formats, output and modify parts of the date object, and to convert the date object to a string.
